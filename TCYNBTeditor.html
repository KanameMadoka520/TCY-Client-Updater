功能说明

工具栏按钮（编辑模式下可用）： 复制 / 剪切 / 粘贴 / 删除 / 重命名 / 添加子节点 / 名称排序 / 类型排序
右键菜单： 在任意节点上右键可快速访问所有编辑操作和"复制为 SNBT"
拖拽排序： 编辑模式下可拖拽同级节点进行排序
多标签页： 同时打开多个文件，每个标签独立编辑状态
HEX 视图： 展开数组节点后点击 HEX 按钮切换十六进制显示
.mca 区域文件： 打开 .mca 文件显示 32×32 区块网格，点击区块查看 NBT 数据
JSON 导出/导入： 将 NBT 数据导出为可读 JSON 格式，或从 JSON 文件导入
拖放文件： 将 .dat / .mca 文件直接拖入窗口即可打开
