ℹ️ 模组来源与恢复指南 (点击展开查看)

1. 模组来源说明：
本客户端模组应当包含以下来源，请勿随意混入来源不明的文件：
  • 基础包自带：Minecraft糖醋鱼神人服务器客户端 20260125 异界生活冒险.zip
  • 或替换包：【新玩家无视本文件】全量mods文件夹替换用-旧客户端升级.zip
  • 以及：本更新器后续自动下载的更新文件。
2. 误删恢复 / 坏档修复 (重要)：
如果您不小心误删了模组，请严格按照以下步骤操作：
  1. 前往 【神人服务器官方网站】 - 【服务器文件下载】 页面。
  2. 下载 【新玩家无视本文件】全量mods文件夹替换用-旧客户端升级.zip。
  3. 解压并将里面的文件 全量覆盖 到您当前的 mods 文件夹中。
  4. 修改更新器同目录下的 launcher_settings.json 文件：
    找到 "current_version" 栏目，将其修改为：
    "current_version": "26.02.06.15.24"
  5. 保存文件后 重启本更新器。更新器将默认您处于初始版本，并逐步获取后续的所有更新模组，您只需依次点击更新即可。

关于 pre-restore 备份（还原前备份）
当你执行“预览还原 → 勾选确认 → 一键还原”时，系统会在真正还原前自动创建一份当前配置的快照，这就是 pre-restore 备份。
它的作用是提供一粒“撤销 / 后悔药”：如果这次还原结果不符合预期，你可以在备份列表里找到名称通常带 pre-restore 的那条备份，再选择它进行还原，即可回滚到还原前的状态。
边界说明：pre-restore 只会仅覆盖本次选择的 config 子目录，以及被选择的 config 根目录文件（如 options.txt），不会动到其它目录/文件（与本页“子范围备份/还原”的规则一致）。
